# MacOS

## Dock

Select/Focus Dock: Ctrl+Fn+F3

## Switching

Applications: Cmd+Tab or Cmd+Shift+Tab

Different Windows Applications: Cmd+`

Tabs within applications: Ctrl+Tab or Cmd+1,2,3...

## Search
